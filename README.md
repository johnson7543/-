下載解壓縮後使用visual studio開啟

需安裝IIS Express方可順利執行

後來才安裝的可能會碰到"Error：Cannot read configuration file"之類的問題

可參考：https://stackoverflow.com/a/49197265/12732471

開啟後對著Views資料夾右鍵->"View in Browser"即可運行網站
